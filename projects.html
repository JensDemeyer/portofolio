Geavanceerd Data Engineering & ML Project

DEP2 - Campus Bezettingsgraad Voorspellingen

Dit project voorspelt hoe druk het is in verschillende lokalen op de campus door het aantal verbonden Wi-Fi-apparaten te vergelijken met de geplande lessen.

Het systeem beheert de volledige data-levenscyclus: van het automatisch ophalen van gegevens via API's tot het opslaan in een robuust centraal gegevensbeheer (Data Warehouse). Kernonderdelen:

  • Geautomatiseerd gegevensbeheer: Dagelijkse processen die netwerkgegevens en roosters verwerken in een SQL Server omgeving.
  • Voorspellingsmodel: Een algoritme dat live voorspelt hoeveel studenten er aanwezig zullen zijn, inclusief uitleg over de belangrijkste factoren.
  • Kwaliteitsbewaking: De volledige code is gecontaineriseerd met Docker en uitvoerig getest op betrouwbaarheid.

Data Engineering & BI Project

DEP1 - Energie Tarieven Vergelijker

Dit systeem vergelijkt automatisch de tariefkaarten van verschillende energieleveranciers om het meest voordelige contract voor elk type gebruiker te vinden.

Hoogtepunten:

  • Automatische Extractie: Informatie ophalen uit complexe PDF-documenten van leveranciers.
  • Gegevensverwerking: De data wordt gestructureerd opgeslagen in een SQL Server database voor snelle analyse.
  • Visualisatie: Gebruiksvriendelijke dashboards in Power BI maken de prijsverschillen direct inzichtelijk.
